Docs

This directory holds project-level reference docs for data contracts and UI/runtime protocols that are larger than a single source file.

Current docs:

  • custom-events.md: Custom DOM event names, emitters, listeners, and detail payloads.
  • effects.md: Runtime effect objects and how they move from engine output to UI behavior.
  • terrain.md: Terrain catalog shape, realm map representations, and editor terrain overrides.
  • editor-architecture.md: Current UI/editor boundaries and the intended component-driven realm editor direction.
  • realm-format.md: Canonical realm JSON shape used by the current runtime.
  • ui-components.md: Guide to current and proposed UI components, with a focus on the realm editor refactor.

Suggested next docs:

  • save-flow.md: Realm save/export flow in play mode and editor mode.
  • entity-model.md: Player, bot, item, and scenery data shapes and behavior.
  • audio-and-effects.md: Sound catalog conventions and how audio relates to effects.
